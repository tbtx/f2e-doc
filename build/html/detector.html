detector¶

客户端信息识别模块，用于自动识别用户使用的客户端环境。

额外提供isIE6属性,isMobile属性，decideMobile(ua)方法来判断mobile类型

  • 硬件设备

  • 操作系统

  • 浏览器

  • 浏览器渲染引擎

    detector.browser.name === “chrome” // true

    // 判断浏览器名方法 2. !!detector.browser.ie // false
